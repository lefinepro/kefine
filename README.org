#+TITLE: OKR Task Dashboard
#+DATE: 2026-02-20
#+STARTUP: overview

* Overview
OKR-focused task dashboard built with Svelte 5, Vite, TypeScript, and TAC (Token-Atom-Component) styles methodology.

* Quick Start
#+BEGIN_SRC shell
cd kefine
bun install
bun run dev
#+END_SRC

* Documentation
- [[tasks/INDEX.org][Task Index]]
- [[tasks/OKR-001-data-model.org][Data Model]]
- [[tasks/OKR-002-store-implementation.org][Store Implementation]]

* Features
- OKR tracking with progress visualization
- Task management with OKR linking
- Advanced task creation (Shift+Enter)
- Repository and file attachments
- Keyboard navigation
- Responsive design

* Tech Stack
- **Frontend:** Svelte 5
- **Language:** TypeScript (strict mode)
- **Build:** Vite
- **Package Manager:** Bun
- **Linting:** Oxlint
- **Styling:** TAC (Token-Atom-Component) CSS methodology

* Scripts
| Command | Description |
|---------+-------------|
| ~bun run dev~ | Start development server |
| ~bun run build~ | Build for production |
| ~bun run preview~ | Preview production build |
| ~bun run lint~ | Run Oxlint |
| ~bun run lint:fix~ | Run Oxlint with auto-fix |
| ~bun run check~ | Run TypeScript type check |
