#+TITLE: OKR Task Dashboard
#+DATE: 2026-02-20
#+STARTUP: overview

* Overview
OKR-focused task dashboard built with Svelte, Vite (Rolldown), and TAC (Token-Atom-Component) styles methodology.

* Quick Start
#+BEGIN_SRC shell
cd okr-task-dashboard
pnpm install
pnpm dev
#+END_SRC

* Documentation
- [[okr-implementation-plan.org][Implementation Plan]]
- [[okr-tasks.org][Task Breakdown]]

* Features
- OKR tracking with progress visualization
- Task management with OKR linking
- Advanced task creation (Shift+Enter)
- Repository and file attachments
- Keyboard navigation
- Responsive design

* Tech Stack
- **Frontend:** Svelte 5
- **Build:** Vite with Rolldown
- **Styling:** TAC (Token-Atom-Component) CSS methodology
- **Package Manager:** pnpm
